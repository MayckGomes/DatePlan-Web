3. Dados Coletados

3.1 Dados fornecidos pelo usuário

  • Nome
  • Endereço de e-mail
  • Compromissos
  • Tarefas
  • Lembretes
  • Datas especiais
  • Informações inseridas voluntariamente pelo usuário para utilização dos recursos disponibilizados pelo aplicativo

3.2 Dados técnicos

  • Modelo do dispositivo
  • Sistema operacional
  • Versão do aplicativo
  • Relatórios de falhas
  • Informações de desempenho e estabilidade da aplicação

Essas informações são utilizadas exclusivamente para diagnóstico técnico, correção de erros e aprimoramento da experiência do usuário.

3.3 Dados que não coletamos

  • Fotografias
  • Áudio ou gravações de voz
  • Dados de localização em tempo real
  • Informações de contatos do dispositivo
  • Dados biométricos
  • Informações sensíveis definidas pela legislação aplicável

4. Finalidade do Tratamento dos Dados

  • Criar e gerenciar contas de usuário
  • Permitir o uso das funcionalidades do aplicativo
  • Sincronizar informações entre dispositivos
  • Garantir a segurança da conta
  • Corrigir falhas técnicas
  • Monitorar estabilidade e desempenho da aplicação
  • Cumprir obrigações legais e regulatórias

O DatePlan não utiliza dados pessoais para publicidade direcionada, venda de dados ou perfilamento comercial de usuários.

5. Bases Legais para o Tratamento

  • Execução dos serviços solicitados pelo usuário
  • Cumprimento de obrigações legais
  • Exercício regular de direitos
  • Legítimo interesse para manutenção da segurança, integridade, estabilidade e melhoria dos serviços

6. Compartilhamento de Dados

O DatePlan não vende, aluga ou comercializa dados pessoais.

  • Com provedores de infraestrutura necessários para operação do serviço
  • Quando exigido por lei
  • Mediante ordem judicial ou solicitação de autoridade competente

7. Serviços de Terceiros

  • Google Firebase Authentication
  • Google Firebase Crashlytics
  • Google Firebase Performance Monitoring
  • Serviços de hospedagem, banco de dados e infraestrutura utilizados pela aplicação

8. Transferência Internacional de Dados

Alguns provedores utilizados pelo DatePlan podem armazenar ou processar informações em servidores localizados fora do país de residência do usuário.

Quando aplicável, essas transferências observam mecanismos reconhecidos pela legislação pertinente, incluindo cláusulas contratuais padrão (Standard Contractual Clauses – SCCs) ou outras salvaguardas adequadas.

9. Autenticação e Sessões

O DatePlan utiliza mecanismos de autenticação e identificadores de sessão necessários para manter o acesso seguro à conta do usuário.

10. Armazenamento e Retenção dos Dados

Os dados serão mantidos enquanto a conta permanecer ativa e forem necessários para prestação dos serviços.

  • Os dados serão removidos ou anonimizados dos sistemas ativos após exclusão da conta
  • Informações podem permanecer temporariamente em backups
  • Dados podem ser preservados quando exigido por lei

11. Segurança das Informações

  • Utilização de conexões protegidas por HTTPS/TLS
  • Controle de acesso aos sistemas
  • Monitoramento de falhas e incidentes
  • Atualizações de segurança
  • Boas práticas de desenvolvimento seguro

Embora sejam adotadas medidas adequadas de proteção, nenhum sistema pode garantir segurança absoluta contra todos os riscos existentes.

12. Incidentes de Segurança

Em caso de incidente de segurança que possa acarretar risco ou dano relevante aos usuários, o DatePlan adotará as medidas cabíveis e realizará as comunicações exigidas pela legislação aplicável.

13. Direitos do Usuário

  • Confirmação da existência de tratamento de dados
  • Acesso aos dados pessoais
  • Correção de dados
  • Exclusão de dados pessoais
  • Portabilidade dos dados
  • Informações sobre compartilhamento de dados
  • Oposição ao tratamento nos casos previstos em lei

Grande parte desses direitos pode ser exercida diretamente através das funcionalidades do aplicativo.

14. Exclusão da Conta

O usuário poderá solicitar a exclusão da conta diretamente através das funcionalidades disponibilizadas pelo aplicativo.

Caso encontre dificuldades, poderá entrar em contato através do e-mail informado nesta Política.

15. Alterações nesta Política

Esta Política poderá ser atualizada periodicamente para refletir alterações legais, regulatórias, técnicas ou operacionais.

Sempre que ocorrerem alterações relevantes, os usuários serão informados através do aplicativo, do site oficial ou por outros meios apropriados.
